# **Настроить IS-IS для Underlay сети.**


## **Цель:**
1. Настроите IS-IS в Underlay сети, для IP связанности между всеми сетевыми устройствам.
2. Зафиксировать план работы, адресное пространство, схему сети, конфигурацию устройств.
3. Диагностика связанности между устройствами в IS-IS домене


### **План работ:**
1. Собираем схему, распредаеляем адресное пространство.
2. Включаем процесс is-is на линковых адресах, а также loopback-ов всех устройств схемы.
3. Диагностика соседства is-is, таблица маршрутизации, доступность всех loopback-ов устройств схемы.
4. Фиксация результатов.
