# **Настроить OSPF для Underlay сети.**


## **Цель:**
1. Настроите OSPF в Underlay сети, для IP связанности между всеми сетевыми устройствам.
2. Зафиксировать план работы, адресное пространство, схему сети, конфигурацию устройств.
3. Диагностика связанности между устройствами в OSFP домене


### **План работ:**
1. Собираем схему, распредаеляем адресное пространство.
2. Включаем процесс ospf на линковых адресах, а также loopback-ов всех устройств схемы.
3. Диагностика соседства ospf, таблица маршрутизации, доступность всех loopback-ов устройств схемы.
4. Фиксация результатов.
